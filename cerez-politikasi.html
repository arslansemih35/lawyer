Çerez Politikası

Son Güncelleme: 15 Ocak 2025

1. Çerez Nedir?

Çerezler (cookies), web sitemizi ziyaret ettiğinizde tarayıcınız aracılığıyla cihazınıza kaydedilen küçük metin dosyalarıdır. Bu dosyalar, web sitesinin daha iyi çalışmasını sağlar ve kullanıcı deneyimini iyileştirir.

2. Çerezlerin Kullanım Amacı

Web sitemizde çerezler aşağıdaki amaçlarla kullanılmaktadır:

  • Zorunlu Çerezler: Web sitesinin temel işlevlerinin çalışması için gerekli
  • Performans Çerezleri: Site performansını analiz etmek ve iyileştirmek için
  • Fonksiyonel Çerezler: Kullanıcı tercihlerini hatırlamak için
  • Analitik Çerezler: Site kullanım istatistiklerini toplamak için

3. Kullandığımız Çerez Türleri

3.1 Zorunlu Çerezler

Bu çerezler web sitesinin temel işlevlerinin çalışması için gereklidir ve devre dışı bırakılamaz:

  • Session Çerezleri: Oturum bilgilerini saklar
  • Güvenlik Çerezleri: Güvenlik kontrollerini sağlar

3.2 Analitik Çerezler

Bu çerezler site kullanımını analiz etmek için kullanılır:

  • Google Analytics: Ziyaretçi istatistiklerini toplar
  • Sayfa Görüntüleme: Hangi sayfaların ziyaret edildiğini takip eder

3.3 Fonksiyonel Çerezler

Bu çerezler kullanıcı deneyimini iyileştirmek için kullanılır:

  • Dil Tercihi: Seçilen dil ayarını hatırlar
  • Form Verileri: Form doldurma sürecini kolaylaştırır

4. Üçüncü Taraf Çerezler

Web sitemizde aşağıdaki üçüncü taraf hizmetler kullanılmaktadır:

  • Google Fonts: Web fontları için
  • Font Awesome: İkonlar için
  • Google Analytics: Site analizi için

Bu hizmetler kendi çerez politikalarına sahiptir ve kullanım koşulları ilgili şirketlerin web sitelerinde bulunabilir.

5. Çerez Yönetimi

Çerezleri yönetmek için aşağıdaki seçenekleriniz bulunmaktadır:

5.1 Tarayıcı Ayarları

Tarayıcınızın ayarlarından çerezleri devre dışı bırakabilir veya silebilirsiniz:

  • Chrome: Ayarlar > Gizlilik ve Güvenlik > Çerezler
  • Firefox: Ayarlar > Gizlilik ve Güvenlik > Çerezler
  • Safari: Tercihler > Gizlilik > Çerezler
  • Edge: Ayarlar > Çerezler ve site izinleri

5.2 Çerez Onayı

Web sitemizi ziyaret ettiğinizde, çerez kullanımına onay verdiğiniz kabul edilir. Çerezleri devre dışı bırakırsanız, sitenin bazı özellikleri düzgün çalışmayabilir.

6. Çerez Süreleri

Çerezlerin saklanma süreleri türlerine göre değişmektedir:

  • Session Çerezleri: Tarayıcı kapatıldığında silinir
  • Kalıcı Çerezler: Belirlenen süre boyunca saklanır (genellikle 1-2 yıl)
  • Analitik Çerezleri: 26 ay boyunca saklanır

7. Çerez Güvenliği

Çerezleriniz güvenli bir şekilde saklanmaktadır:

  • HTTPS protokolü kullanılarak şifrelenmiş iletişim
  • Güvenlik çerezleri ile koruma
  • Düzenli güvenlik güncellemeleri

8. Çerez Politikası Güncellemeleri

Bu çerez politikası, yasal gereklilikler veya teknolojik değişiklikler nedeniyle güncellenebilir. Önemli değişiklikler web sitemizde duyurulacaktır.
